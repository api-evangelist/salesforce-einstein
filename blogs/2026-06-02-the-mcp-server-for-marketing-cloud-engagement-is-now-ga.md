Salesforce has released a generally available MCP (Model Context Protocol) server for Marketing Cloud Engagement, enabling AI agents to manage marketing tasks through plain language commands. The server allows agents to handle data extensions, journeys, automations, and other core marketing functions without requiring code or UI navigation. Setup involves registering an MCP URL with your preferred agent and authenticating through OAuth, with permissions controlled by both the installed package scopes and individual user permissions.
